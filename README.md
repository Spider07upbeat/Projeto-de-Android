# Hub de Jogos

## Desenvolveu-se este projeto, no âmbito da cadeira de Desenvolvimento de Jogos para Plataformas Móveis. O projeto é composto por um hub geral onde se pode selecionar 1 entre 3 jogos, PokeClicker, Nonogram e Sudoku, todos com os dados mais importantes guardados no Firebase.

## O Hub

### PokeClicker
### Este jogo foi desenvolvido, por um dos membros do grupo, como projeto de aula. É um jogo no estilo "Clicker", tal como o nome indica onde o objetivo é eliminar o máximo de pokemons que vão aprecendo na tela e à medida que aumenta o score, altera o background. Possui ainda uma lista de High-Score, entre os 20 jogadores com os maiores scores adquiridos.

### Nonogram
### Este jogo é uma recriação do famoso jogo "Nonogram", em que nos é fornecida uma grelha com indicações nas linhas e colunas e temos de a tentar completar, assim formando uma imagem, e no nosso jogo não é diferente. Temos um Menu Inicial com 3 botões, "Play, para iniciar o jogo, "Back to Main Menu", para voltar ao Hub e por fim "Nonogramas Feitos", que mostra os Nonogramas realizados pelo utilizador nesta conta, através do Firebase.

### Sudoku
### Este jogo é um recriação mais simplificada do jogo "Sudoku", em que seleciona-mos uma dificuldade desejada e após isso é nos fornecida uma grelha 9x9 com alguns números e temos de completar todas as linhas e colunas sem criar repetições. Ao selecionar o botão "Sudoku" o utilizador é direcionado para um menu com 3 botões, sendo eles "Jogar" que nos leva para a seleção de dificuldade para depois completarmos o Sudoku, o botão "Melhores Tempos", que mostra o melhor tempo do utilizador em cada dificuldade e por fim o botão "Voltar ao Hub" que nos leva de volta ao Hub de jogos.

# Hub de Jogos

## Desenvolveu-se este projeto, no âmbito da cadeira de Desenvolvimento de Jogos para Plataformas Móveis. O projeto é composto por um hub geral onde se pode selecionar 1 entre 3 jogos, PokeClicker, Nonogram e Sudoku, todos com os dados mais importantes guardados no Firebase.

## O Hub


### Menu Inicial
### O projeto inicia numa página de Login, requisitando email e password para aceder ao Hub de jogos, caso o utilizador não tenho conta cria pode criar uma no botão embaixo do Login "Não tem conta? Registe-se aqui"

### PokeClicker
### Este jogo foi desenvolvido, por um dos membros do grupo, como projeto de aula. É um jogo no estilo "Clicker", tal como o nome indica onde o objetivo é eliminar o máximo de pokemons que vão aprecendo na tela e à medida que aumenta o score, altera o background. Possui ainda uma lista de High-Score, entre os 20 jogadores com os maiores scores adquiridos.

### Nonogram
### Dentro do jogo, o jogador tem de apontar, alterar o ângulo, a força do disparo e por fim disparar uma bolha de água para eliminar os slimes inimigos. À medida que o jogador elimina inimigos ele progride nos níveis onde a dificuldade aumenta(A velocidade dos inimigos aumenta) e aumenta o seu score, que será adicionado ao High Score quando o jogador perder todas as suas 5 vidas, que são perdidas ao falhar os disparos.
